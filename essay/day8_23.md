## es6相关

+ 直接执行的语句

es6以前是这么写

    （function fn(){
        conosle.log(111111111111);
    }()）;

es6中有

    {
        var x1 = 1;
        console.log(111111111);
    }

下面这个代码在es6中会报错，原因在于es6中的let会导致暂时性锁区

    var a = 1;
    {
        console.log(a);
        let a = 2;
        console.log(a);
    }

es6函数设置默认参数

    function fn(a = 3,b = 4){
        console.log(a + ","+ b)
    }
    fn()  //3,4
    fn(1) //1,4
    fn(undefined,3); //3,4 (只有当设为undefined时会略过第一个参数)
